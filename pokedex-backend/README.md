# POKEDEX | back-end
Api, Pokedex, que consome a pokeapi.

## Stack
- Java 17
- Gradle
- SpringBoot;
- SpringDataJPA;
- Postgres.

## Configuração
- Criar um database com o nome "pokemons".
- Configurar o arquivo src/main/resources/application.properties para as credenciais do banco local.
- IDE utilizada: IntelliJ.
